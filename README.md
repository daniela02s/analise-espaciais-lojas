# Análise Espacial de Lojas com Python

Este repositório é dedicado ao estudo de **Análise Espacial** e **Data Visualization**. O projeto foca em identificar padrões geográficos, distribuição de pontos de venda e a utilização de técnicas avançadas de estilização de dados para transformar coordenadas e informações brutas em insights estratégicos.

---

## Objetivos do Projeto

Neste projeto, aplico os conceitos fundamentais de visualização e manipulação de dados aprendidos no curso de **Data Visualization**:

- **Estilização de Tabelas Geográficas**: Uso da classe `Styler` do Pandas para apresentar dados de localização de forma legível.
- **Built-in Styles**: Aplicação de estilos nativos para identificar rapidamente lojas com maior faturamento ou fluxo de clientes.
- **Customização com CSS**: Uso de propriedades CSS para criar tabelas profissionais que acompanham mapas e relatórios.
- **Análise de DataFrames**: Estruturação de dados espaciais para facilitar a tomada de decisão sobre expansão e localização.
- **Apresentação de Informações**: Construção de visualizações adaptadas para cenários de geomarketing.

## Tecnologias e Bibliotecas

Para rodar as análises e visualizar as tabelas estilizadas, são utilizadas as seguintes ferramentas:

- **Python 3.x**
- **Pandas**: Manipulação de dados e estilização (`df.style`).
- **Geopandas**: Para manipulação de dados espaciais e coordenadas.
- **Jinja2**: Necessário para a renderização de HTML/CSS das tabelas.
- **Matplotlib/Seaborn**: Para geração de gráficos e mapas de calor.
- **Folium/Plotly**: (Opcional) Para mapas interativos.
